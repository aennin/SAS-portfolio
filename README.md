# SAS Programming Portfolio

This portfolio showcases SAS programming skills developed through graduate coursework at Western Michigan University (STAT 6800: SAS Programming).

## Skills Demonstrated
- **Data Step Programming**: Data import, cleaning, transformation, merging
- **PROC SQL**: Complex queries, joins, subqueries, data summarization
- **Macro Programming**: Automated reporting, dynamic code generation
- **ODS & Reporting**: HTML, PDF, RTF output with custom formatting
- **Statistical Procedures**: PROC MEANS, PROC FREQ, PROC REG, PROC ANOVA

## Projects
1. **Data Management Project**: Cleaning and merging multiple datasets
2. **Reporting Automation**: Macro-driven report generation
3. **Statistical Analysis**: Regression modeling and hypothesis testing
